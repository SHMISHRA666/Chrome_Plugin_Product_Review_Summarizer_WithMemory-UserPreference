Troubleshooting Tips

  1. Make sure the backend server is running with:
    python mcp_client.py --server
  2. Verify the server is running on port 8080
  3. Check for any errors in the terminal where the server is running
  4. Ensure the API endpoint is correct:
    http://localhost:8080/api/detect-product
  5. If you've changed the server port or URL, update it in the extension settings
